# gatsby-plugin-segment

A [Gatsby](https://www.gatsbyjs.org) plugin for [Segment](https://segment.com/).

## Usage

In your `gatsby-config.js`, add:

```js
module.exports = {
  plugins: [
    // your other plugins...
    {
      resolve: 'gatsby-plugin-segment',
      options: {
        writeKey: 'YOUR WRITE KEY'
      }
    }
  ]
}
```
